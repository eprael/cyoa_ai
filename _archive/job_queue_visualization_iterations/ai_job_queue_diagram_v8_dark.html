01 User clicks Generate Story in the browser Browser
02 Browser sends a POST request to the PHP web app Browser → PHP
03 PHP inserts a row into ai_jobs with status pending and returns immediately PHP → DB
04 Every 4 seconds, the browser polls PHP: "is my job done yet?" Browser → PHP → DB
05 Every 30 seconds, the cron worker wakes up and scans for pending jobs Cron → DB
06 Worker marks the job running, then calls the appropriate AI API Cron → Cloud
07 AI API returns result; worker stores it and marks job completed Cloud → Cron → DB
08 Browser's next poll finds status = completed and displays the result Browser ← PHP ← DB
